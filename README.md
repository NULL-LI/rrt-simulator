# RRT Simulator

## Informed-RRT-STAR algorithm

First using rrt-connect to find a way to go. 

After that informed-rrt-star is used for optimization.

## Dependencies
* Qt5
* Eigen

## Compiling
```bash

$ cd rrt-simulator/
$ mkdir build
$ cd build
$ cmake..
$ make -j

```
Run the exectuable as
```
$ ../bin/rrt-test
```
